# ZhiYeCompass Project

**智业罗盘** - AI副业/创业推荐平台 (中国用户)

## Stack
Next.js 14 + shadcn/ui + AWS Lambda (ca-central-1) + DynamoDB + S3

## Key Features
- No login required (Session Storage + Cookie UUID)
- LLM dynamic project generation (no pre-built database)
- Hourly quota control
- One recommendation at a time
- Risk-focused (legal, financial, platform, competition)
